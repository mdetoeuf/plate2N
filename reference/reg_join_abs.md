# Merge Data from Regression Equation on Standard Curve with Absorbance Data (samples)

Merge Data from Regression Equation on Standard Curve with Absorbance
Data (samples)

## Usage

``` r
reg_join_abs(lm_table, abs_data, target_sp = "N")
```

## Arguments

- lm_table:

  A tibble with linear model data (as generated by
  [`lm_std_curve()`](https://mdetoeuf.github.io/plate2N/reference/lm_std_curve.md)).
  Must contain columns `plate_id`, `slope`, `adj_r_squared`, `lm_p`

- abs_data:

  Blank-corrected absorbance data, in a similar format to tidy_plates,
  but containing only sample wells, no empty wells, standard curve wells
  or extractant wells

- target_sp:

  Defaults to "N", the target compound to convert to

## Value

A tibble joining linear model data with absorbance data

lm_table <- lm_output$lm_data
abs_data <- abs_corrected
