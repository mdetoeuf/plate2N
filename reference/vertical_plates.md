# Example of the verticalized plate data

This data set was generated by the
[`verticalize_plates()`](https://mdetoeuf.github.io/plate2N/reference/verticalize_plates.md)
function, ran on our example data set
[`tibble_example`](https://mdetoeuf.github.io/plate2N/reference/tibble_example.md).

## Usage

``` r
vertical_plates
```

## Format

A tibble with 96 rows (1 per well of the 96-well plate) and 6 columns

- row:

  1-letter identifier of the original plate-row (letters A to H)

- column:

  1-number identifier of the original plate-column (numbers 1 to 12)

- NO3_TDN_01 to NO3_TDN_04:

  each column corresponds to 1 plate of the input tibble and contains
  verticalized plate data
