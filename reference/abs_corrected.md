# Example of Blank-corrected Absorbance Data (samples)

Example of Blank-corrected Absorbance Data (samples)

## Usage

``` r
abs_corrected
```

## Format

Tibble in format as generated by
[`blank_correct_abs()`](https://mdetoeuf.github.io/plate2N/reference/blank_correct_abs.md)

- abs_corrected:

  blank-corrected absorbance

- other columns:

  Same columns as `tidy_plates`, + extractant average
